# Jet's Pizza Homepage Clone (local demo)

Static clone of the Jet's Pizza marketing homepage for local preview.

## Run

```bash
cd jets-pizza-clone
python3 -m http.server 5173 --bind 127.0.0.1
```

Open http://127.0.0.1:5173/

## Screenshots / PDF

```bash
npm install
npx playwright install chromium
node capture.mjs
python3 make_pdf.py
```

Preview PDF: `artifacts/JetsPizza_Clone_Preview.pdf`
